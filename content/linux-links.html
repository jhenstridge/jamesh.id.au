Distributors

  • Redhat - The distribution we use. You can download most of it from their ftp site.
  • Debian - A truely free distribution.
  • Slackware - a popular, but less complex distribution.

Web Related

  • Web Servers
    • Apache Web Server - The most popular. It has the largest market share, and the installation base keeps on increasing. There are also a number of side projects associated with it:
      • Apache Week Mailing List
      • Apache/Perl Integration Project
      • Apache/Java Integration Project
    • NCSA Web Server - Old. Use Apache.
    • Roxen Challenger - These people give away a good web server, but I haven't seen any speed tests against Apache.
  • Proxy Servers
    • Squid Proxy Cache

Database Systems

  • SQL Servers
    • MySQL SQL Server - This is a powerful SQL engine, and you don't have to pay for it unless you sell a product that uses it.
    • mSQL Server - This is the product that MySQL was written to look like, but is inferior to MySQL in a number of ways.
    • PostgreSQL - This one is full of features. For some people it might be overkill.
  • SQL Applications
    • WWW-SQL - A very useful utility to put data from a MySQL database on a Web page, on the fly. It just so happens to be written by me.

Development

  • Free Software Foundation - This is the association responsible for a lot of the software that comes with linux.
  • Python - A programming language that is extendable, object orriented, and has facilities to do platform independant GUIs. Also one of my favourite languages.
  • Perl - Very useful.
  • TCL/TK - A horrible language (IMO), but you should install it, since their is a python interface to its GUI toolkit.
